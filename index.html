Soundcloud visualizer: a canvas and webaudio API experiment by Michael Bromley.
For more information, visit the project's Github page

Instructions
Paste the URL of a Soundcloud track or playlist into form and press enter.

Example track: https://soundcloud.com/manchesterorchestra/simple-math
Example playlist: https://soundcloud.com/manchesterorchestra/sets/simple-math-album-stream

Controls
space bar - toggle play/pause
right/left arrow - skip forward/backwards through playlist
